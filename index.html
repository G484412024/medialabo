各授業回の学習事項

第1回 WWW と HTML

WebページはHTMLとCSS、JavaScript、画像・音声・動画によって構成されている。HTMLには要素と属性があり、さらに要素にはブロック要素とインライン要素がある。属性とは要素に追加する付加情報である。段落やメニューなどがブロック要素で、強調やリンクなどはインライン要素に当たる。インライン要素の中にブロック要素は配置できない。HTMLには特殊文字が5つある。

演習の成果: 演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

CSSによるHTMLの装飾の仕方、HTMLにどのようにCSSを追加するのか、CSSの調べ方やデバックのやり方について学んだ。さらにCSSでは、文字の大きさや色、書体、特定の要素を指定、文字の裏に背景色、画像やレイアウトなども詳細に設定できることを学んだ。

演習の成果: 演習2-3:ウェブブラウザ説明ページの装飾

第3回 JavaScript

JavaScriptはwebページ上で複雑な機能を実現するためのプログラミング言語だと理解した。JavaScriptをページに反映させる方法を学んだ。コンソールにてプログラムを確認し、VSに書き写すという手順を学んだ。オブジェクトの作り方や扱いについて学んだ。開発者ツールを使ったデバッグのやり方について学んだ。

演習の成果: 演習3-6:オブジェクトの参照

第4回 DOM と JavaScript

ボタンを押したらページが変化する処理をJavaScriptプログラムとして記述する方法を学んだ。入力欄を作るinput要素とボタン、input要素のtype属性について学んだ。要素を検索し、参照する方法を学んだ。また、ユーザが設定した値を参照し、要素のテキストを変更することによって、ユーザの設定した値で結果を表に表示させる方法について学んだ。イベントハンドラの登録と確認の仕方について学んだ。

演習の成果: 演習4-10:足し算ページ

第5回 イベント処理

DOMの要素の新規作成、要素の編集(要素の属性の追加、変更)、要素の削除の使い方について学んだ。

演習の成果: 演習5-7:DOMプログラミング(2)
